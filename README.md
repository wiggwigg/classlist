# Classlist Builder

Single-page web app that transforms an Allocate/Excel student export into:

- One master tab (unit code) with internal links from `Student ID` to class tabs
- One tab per class (for example, `Tutorial 01`)
- Sorted lists by `Last name`, `First Name`, `Student ID`

## Run locally

Open `index.html` in a browser.

## Privacy

Processing runs in your browser session only. No data is uploaded or stored by this app.
